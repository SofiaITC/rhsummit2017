buildansibletower
=========

This role builds ansible tower and is step 1 in building the redhat management suite. 

Requirements
------------

- Ansible 2.2.1
- Ansible vault file in group_vars/all/vault
- Ansible Tower License
- Satelite 6 Manifest
- AWS cli
- Private key for ec2 instances
- CFME image for AWS
